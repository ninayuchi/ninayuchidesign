Nina Ment

Web design, web development



I appropriated the standard format of an online shopping site to create one that “sells” the random pieces of trash discarded around my dorm. I was influenced by online brands that rely solely on extravagant marketing and flamboyant websites to attempt authenticity and distract from their mediocre products, so this website contains several flaws (the server not working, only a few items, no reviews).

This website was created with HTML, CSS, jQuery, and JavaScript for the Web Design wintersession course. Nina Ment was selected for and displayed at the RISD 2018 Graphic Design Triennial.
